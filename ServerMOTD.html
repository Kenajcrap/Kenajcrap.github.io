Server Experimental

Criei este server para aprender um pouco mais sobre LUA e coding como um todo, por isso: Não esperem suporte aos players ou uma administração ativa.

Na verdade, nao esperem nem que o gamemode permaneça esse, no momento estou trabalhando com Deathrun, mas isso pode mudar no futuro!

Como Jogar Deathrun

Bem vindo ao deathrun, existem 2 objetivos:

  • Como Death, impeça que os runners cheguem no fim do mapa a qualquer custo.
  • Como Runner, evite as traps e chegue no final do mapa para derrotar os Death.

Basicos

  • Se você for o unico no servidor você será permitido de explorar o mapa em Godmode (você não levará dano). Digite !r para se renascer, !cleanup para resetar as traps, e !rtv para trocar o mapa.
  • Este server possui uma função de pulo automático. Usar o "autojump" restringe sua velocidade a um valor espefícico. Para evitar isso, desative o "autojump" no menu F2
  • Escreva !info ou aperte F1 para reabrir essa janela
  • Obedeça as regras do server, elas estão aqui pra isso
  • Seja legal com os outros jogadores, nao quero ter que banir ninguém.
